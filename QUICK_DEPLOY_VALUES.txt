 🗄️ MONGODB ATLAS
═══════════════════════════════════════════════════════════════════

URL: https://www.mongodb.com/cloud/atlas

Steps:
1. Create Free Cluster (M0)
2. Create Database User (save password!)
3. Network Access → Allow from Anywhere (0.0.0.0/0)
4. Get Connection String

Connection String Format:
mongodb+srv://USERNAME:PASSWORD@cluster0.xxxxx.mongodb.net/investments?retryWrites=true&w=majority

(Replace USERNAME and PASSWORD with your actual values)
